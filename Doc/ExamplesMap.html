Getting started: how to find the relevant example

Metamodeling features

  1. Kriging (gaussian process based surrogate model)
    • Kriging_Branin.m: build a Kriging metamodel of the Branin test function
    • Error_prediction_Branin.m: reduce the Kriging prediction error on the Branin test function with MSE criterion

  2. Radial Basis Function (RBF-based surrogate model)
    • RBF_Branin.m: build a RBF metamodel of the Branin test function

  3. Co-kriging (multifidelity metamodel based on Kriging)
    • Cokriging_ForresterTest.m: build a Kriging multifidelity metamodel of the Forrester test example from low and high fidelity numerical models

  4. Co-RBF (multifidelity metamodel based on RBF)
    • CoRBF_ForresterTest.m: build a RBF multifidelity metamodel of the Forrester test example from low and high fidelity numerical models

  5. Kriging for mixed problemes (BQQV, both quantitative and qualitative variables)
    • Kriging_Qualitative.m: build a metamodel with mixed parameters (example with 1 quantitative variable and 1 qualitative variable with 3 levels)

Adaptive Sampling strategy

  1. Adaptive sampling using a Kriging metamodel (efficient global optimization of the Expected improvement)
    • EGO_unconstrained.m: optimization of the Branin test problem with an adaptive sampling algorithm, no constraints
    • EGO_constrained.m: constrained optimization of the Branin test problem with an adaptive sampling algorithm
    • EGO_cheap_constraint.m: constrained optimization of the Branin test problem with an adaptive sampling algorithm, one additional constraint computed with an analytical expression
    • EGO_unconstrained_qualitative.m: optimization of a mixed problem with an adaptive sampling algorithm, no constraints
    • EGO_constrained_qualitative.m: constrained optimization of a mixed problem with an adaptive sampling algorithm
    • EI_Cokriging_ForresterTest.m: unconstrained optimization of a multifidelity problem (1D Forrester test function) with an adaptive sampling algorithm
    • EI_Cokriging_Branin.m: unconstrained optimization of a multifidelity problem (2D Branin test function) with an adaptive sampling algorithm

  2. Adaptive sampling using a RBF-based metamodel
    • CORS_Branin_unconstrained.m: optimization of the Branin test problem with an adaptive sampling algorithm and the CORS criterion
    • Gutmann_Branin.m: constrained optimization of the Branin test problem with an adaptive sampling algorithm and the Gutmann criterion
    • Gutmann_cheap_constraint.m: constrained optimization of the Branin test problem with an adaptive sampling algorithm and the Gutmann criterion, one additional constraint computed with an analytical expression

  3. Multiobjective optimization
    • Multiobjective_EHVI.m: multi-objective optimization of a numerical model with the Expected Hypervolume Improvement.
    • Multiobjective_EI_MGDA.m: multi-objective optimization of a numerical model with the MOEGO strategy (multi-gradient descent on the expected improvement)
    • Multiobjective_EI_MGDA_qualitative.m: multi-objective optimization of a mixed problem with the MOEGO strategy (multi-gradient descent on the expected improvement)

  4. Robust efficient global optimization
    • Robust_EGO_1D.m: optimization of the output of a 1D model when inputs parameters have some uncertainties
    • Robust_EGO_2D_cons.m: constrained optimization of the output of a 2D model when inputs parameters have some uncertainties
    • Robust_EGO_multiobj_2D.m: two-objective optimization of the output of a 2D model when inputs parameters have some uncertainties

Optimization features and misc. tools

  1. Optimisation methods
